Feature A - Grow 2

The flex-grow: 2 property allows this box to take twice the extra space compared to others when there is free room in the container.

Feature B - Balanced

This box uses flex: 1 1 150px which means it grows and shrinks at normal rate and starts from a base of 150px.

Feature C - Shrink Faster

With flex-shrink: 2, this box will shrink twice as fast when there's not enough space. It helps prioritize other boxes.

Feature D - Smallest

This box starts from 50px flex-basis and has flex-shrink: 3, so it gets squeezed first on small screens.
